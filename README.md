## 💻 대회주제
- **주제: 인터넷 사용자의 MBTI 예측하기**

- 데이터셋
  - 트레이닝 데이터(MBTI_train.csv): 74,357건
  - 테스트 데이터(MBTI_test.csv): 9,337건
